## Installation Methods

### PyPI Installation

```bash
# Latest version
pip install django-revolution

# Specific version
pip install django-revolution==1.0.13

# With extras
pip install django-revolution[dev]
```

### Poetry Installation

```bash
# Add to project
poetry add django-revolution

# Add development dependencies
poetry add --group dev django-revolution[dev]
```

### Conda Installation

```bash
# Using conda-forge
conda install -c conda-forge django-revolution

# Or using pip in conda environment
conda install pip
pip install django-revolution
```

### Docker Installation

```dockerfile
# Dockerfile
FROM python:3.11-slim

# Install Django Revolution
RUN pip install django-revolution

# Set up your Django project
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt

# Your Django project files
COPY . .
```
